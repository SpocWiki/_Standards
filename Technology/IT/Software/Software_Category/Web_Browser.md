## #has_/text_of_/abstract 

> A web browser, often shortened to browser, is an application for accessing websites. 
> 
> When a user requests a web page from a particular website, 
> the browser retrieves its files from a web server 
> and then displays the page on the user's screen. 
> Browsers can also display content stored locally on the user's device.
>
> Browsers are used on a range of devices, including 
> desktops, laptops, tablets, smartphones, smartwatches, smart televisions and consoles. 
> 
> As of 2024, the most used browsers worldwide are 
> - Google Chrome (~66% market share), 
> - Safari (~16%), 
> - Edge (~6%), 
> - Firefox (~3%), 
> - Samsung Internet (~2%), and 
> - Opera (~2%). 
> 
> As of 2023, an estimated 5.4 billion people had used a browser.
>
> [Wikipedia](https://en.wikipedia.org/wiki/Web%20browser)
