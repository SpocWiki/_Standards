
# [[Structure~Sequence]] 

AKA Block. 
This is just a Sequence of Operations treated as a Unit. 
Could be realized as a named [[Structure~Subroutine|Structure~Subroutine]],
which also provides an Abstraction.

	<tr> <td >>> RETURN Expression(args) >></td></tr>
</table>
