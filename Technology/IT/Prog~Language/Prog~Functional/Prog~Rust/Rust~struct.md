
# [[Rust~struct]] 

Structs are the [[../Algebraic_Data-Types/Product-Type]]s in Rust. 
Their Size is the Sum of the Constituents' Sizes. 
Unlike [[Rust~Tuple]]s their Members are named. 

## Syntax 

### Declaration
```rust
struct Move { x: i32, y: i32 }
```

### Instantiation
```rust
Move { x: 10, y: 30 },
```
