# [[False_Memory]] 


## #has_/text_of_/abstract 

> In psychology,  a **False Memory** is a phenomenon 
> where someone recalls something that did not actually happen 
> or recalls it differently from the way it actually happened. 
> 
> Suggestibility, activation of associated information, the incorporation of misinformation, 
> and source misattribution have been suggested to be 
> several mechanisms underlying a variety of types of false memory.
>
> [Wikipedia](https://en.wikipedia.org/wiki/False%20memory) 


It is proven that traumatic Memories are not hidden. 
It is much more probable that false Memories are created by Suggestion.
