# Geraniales 

![Erodium cicutarium](Erodium_cicutarium2596.jpg)

-   *Melianthaceae* [(incl. Francoaceae and Greyiaceae)]
-   *Geraniaceae* [(geranium family, incl.
    Hypseocharitaceae)]
-   *Vivianiaceae* [(incl. Ledocarpaceae)]

Containing group: [Rosids](Rosids.md)
