# [[Function~ternary]] 

The most common ternary Function is the `?:` Operator or its equivalend `if/else` structure. 
It is not an Operator, because the first Argument is a [[../../schema-org/Class/is_a_/Data_Type/Boolean|Boolean]] Value. 

## #has_/text_of_/abstract 

> In mathematics, a **Ternary operation** is an n-ary operation with [[../../schema-org/Arity|Arity]] = 3. 
> A ternary operation on a set A takes any given three elements of A 
> and combines them to form a single element of A.
>
> In computer science, 
> a ternary operator is an operator that takes three arguments as input 
> and returns one output.
>
> [Wikipedia](https://en.wikipedia.org/wiki/Ternary%20operation)
