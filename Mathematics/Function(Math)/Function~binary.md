## #has_/text_of_/abstract 

> In mathematics, a **binary function** (AKA bivariate function, or function of two variables) 
> is a function that takes two inputs.
>
> Precisely stated, a function f is binary if there exists sets X,Y,Z 
>  such that f : X×Y → Z where X × Y is the Cartesian product of X and Y. 
>
> [Wikipedia](https://en.wikipedia.org/wiki/Binary%20function)
