[RdfA-Lite is a W3C Standard](http://www.w3.org/TR/rdfa-lite/) to embed RDF Data in HTML using Attributes. 
It is an Alternative to [[Microdata]]

RDFa Lite consists of only five attributes: 
- vocab 
- typeof 
- property 
- resource 
- prefix
