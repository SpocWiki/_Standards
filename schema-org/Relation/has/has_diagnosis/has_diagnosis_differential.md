# [[has_diagnosis_differential]] 

Use it like this: 
- #has_/diagnosis_/differential = `=this.dv_has_diagnosis_differential`  
- has_diagnosis_differential = `=this.dv_has_diagnosis_differential`  

One of a set of differential diagnoses for the condition. 
Specifically, a closely-related or competing diagnosis typically considered later in the cognitive process whereby this medical condition is distinguished from others most likely responsible for a similar collection of signs and symptoms to reach the most parsimonious diagnosis or diagnoses in a patient.
