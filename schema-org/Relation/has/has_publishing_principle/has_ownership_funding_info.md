(often but not necessarily a [[../../../../Society/Agent/Community/Organization/News_Media_Organization|News_Media_Organization]]), 
a description of organizational ownership structure; funding and grants. 

In a news/media setting, 
this is with particular reference to editorial independence.

and can be used to make basic funder information machine-readable.
