#is_a_/tourist_attraction

A tourist attraction.  In principle any Thing can be a [[TouristAttraction]], from a[[landform/mountain]]] and [[LandmarksOrHistoricalBuildings]] to a [[LocalBusiness]].  This Type can be used on its own to describe a general [[TouristAttraction]], or be used as an [[additionalType]] to add tourist attraction properties to any other type.  (See examples below)
