Class of all [[user_interaction]]-Counts.

[[User_Interaction]] and its subtypes are an old way of counting users interacting with pages. 
It is generally better to use [[../Intangible/structured_value/interaction_counter|InteractionCounter]]s with an [[../../../Relation/has/has_interaction_type|interaction-type]]- Relation to the extensible [[Action]] vocabulary, in this case [[user_interaction|user_interaction]], alongside types such as [[Comment]].
