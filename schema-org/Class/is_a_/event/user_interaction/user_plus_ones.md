Class of all [[user_plus_ones]]-Counters.

[[User_Interaction]] and its subtypes are an old way of counting users interacting with pages. 
It is generally better to use [[../Intangible/structured_value/interaction_counter|InteractionCounter]]s with an [[../../../Relation/has/has_interaction_type|interaction-type]]- Relation to the extensible [[Action]] vocabulary, in this case [[../../action/assess_action/react_action/like_action|like-action]], alongside types such as [[Comment]].
