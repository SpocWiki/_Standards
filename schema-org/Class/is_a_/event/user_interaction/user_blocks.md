Class of all [[user_blocks]]-Counters, i.e. Users blocking a Web-Site or Pop-Ups.

[[User_Interaction]] and its subtypes are an old way of counting users interacting with pages. 
It is generally better to use [[../Intangible/structured_value/interaction_counter|InteractionCounter]]s with an [[../../../Relation/has/has_interaction_type|interaction-type]]- Relation to the extensible [[Action]] vocabulary, in this case [[../../action/interact_action/communicate_action/comment_action|CommentAction]], alongside types such as [[Comment]].
