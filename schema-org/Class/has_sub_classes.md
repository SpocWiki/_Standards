# [[has_sub_classes]] 

Indicates a ([[../Collection/item_list|item_list]] of) [[Thing]] is an instance of this [[../Class]].

Use it like this: 
- [ #has_/sub_classes :: [[../Class]] ] or 
- [ has_sub_classes :: [[../Class]] ] 

Relation describes that: 
[ #has_/domain  :: [[../Class]] ]
( #has_/name :: has_sub_classes )
( #has_/range :: [[Thing]] )

But typically it is a typed [[../Collection/item_list|item_list]] with sub-classes. 

#is_/inverse_of  :: [[is_sub_class_of]]
