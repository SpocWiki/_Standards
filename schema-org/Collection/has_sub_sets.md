# [[has_sub_sets]] 

Indicates a ([[../Collection/item_list|item_list]] of) [[../Collection]]s is a subset of this [[../Collection]].

Use it like this: 
- [ #has_/sub_sets :: [[../Collection]] ] or 
- [ has_sub_sets :: [[../Collection]] ] 

Relation describes that: 
[ #has_/domain  :: [[../Collection]] ]
( #has_/name :: has_sub_sets )
( #has_/range :: [[../Collection]] )

But typically it is an example [[../Collection/item_list|item_list]] with sub-sets.
