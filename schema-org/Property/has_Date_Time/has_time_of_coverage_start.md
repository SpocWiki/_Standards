Predicate to describe the time of LiveBlogPosting.

Use it like this: 
- [has_time_of_coverage_start::P#Y#M#W#DT#H#M#s.fff] 
- [ #has_/time_/of_coverage_start::P#Y#M#W#DT#H#M#s.fff] 
with the [ISO_8601-Date Format](../../../ISO/ISO_8601-Date_Time) .

The time when the live blog will begin covering the Event. Note that coverage may begin before the Event's start time. The LiveBlogPosting may also be created before coverage begins.

Formal Predicate: 
[domain::LiveBlogPosting]
(has_/name::has_time_of_coverage_start)
(range::DateTime)
