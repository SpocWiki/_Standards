Predicate to describe the date of DeliveryEvent.

Use it like this: 
- [has_time_available_through::P#Y#M#W#DT#H#M#s.fff] or 
- [ #has_/time_/available_through::P#Y#M#W#DT#H#M#s.fff] with the [ISO_8601-date Format](../../../ISO/ISO_8601-Date_Time) .

After this date, the item will no longer be available for pickup.

Formal Predicate: 
[domain::DeliveryEvent]
(name::has_time_available_through)
(range::DateTime)
