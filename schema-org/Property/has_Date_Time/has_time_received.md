Predicate to describe the date of Message.

Use it like this: 
- [has_time_received::P#Y#M#W#DT#H#M#s.fff] or 
- [ #has_/time_/received::P#Y#M#W#DT#H#M#s.fff] with the [ISO_8601-date Format](../../../ISO/ISO_8601-Date_Time) .
