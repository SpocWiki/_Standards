Predicate to describe the date of Message.

Use it like this: 
- [has_time_read::P#Y#M#W#DT#H#M#s.fff] or 
- [ #has_/time_/read::P#Y#M#W#DT#H#M#s.fff] with the [ISO_8601-date Format](../../../ISO/ISO_8601-Date_Time) .
