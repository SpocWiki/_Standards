Predicate to describe the Text of Person. 

Use it like this: 
- [ #has_/name_/family :: Text ] or 
- [ has_name_family :: Text ] 

Family name. 
In the U.S. and most countries, the last Part of a Persons name. 
In China and many Asian countries the first Part of a Persons name.  

Predicated describes that: 
[ #has_/domain  :: Person ] 
( #has_/name :: has_name_family ) 
( #has_/range :: Text )
